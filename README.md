# Data Analyst Portfolio

A professional Data Analyst portfolio website built with React, Vite, and deployed on GitHub Pages.

## Features
- **Clean, Professional Design**: Muted colors, Inter typography, no distractions.
- **5-Page Structure**: Home, Skills, Experience, Projects, Contact.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **Project Structure**: Component-based React architecture.

## Tech Stack
- **Framework**: React 19 + Vite
- **Styling**: Vanilla CSS (Variables + Design System)
- **Routing**: React Router DOM (HashRouter)
- **Deployment**: GitHub Pages

## Live Site
[View Portfolio](https://isushantk.github.io/Data-analyst-portfolio/)

## Local Development

```bash
# Install dependencies
npm install

# Run dev server
npm run dev
```

## Deployment

```bash
npm run deploy
```
